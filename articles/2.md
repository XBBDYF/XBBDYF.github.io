# CSS 容器查询实战

> 告别视口束缚，让组件真正响应其父容器尺寸。

## 什么是容器查询？

传统的响应式设计依赖 `@media` 查询，基于**浏览器视口**宽度来调整布局。而**容器查询**允许我们根据**某个父元素容器**的尺寸来应用样式，让组件更加模块化、可复用。

```css
/* 定义容器 */
.card-container {
  container-type: inline-size;
  container-name: card;
}

/* 当容器宽度 ≥ 400px 时 */
@container card (min-width: 400px) {
  .card {
    display: flex;
    flex-direction: row;
  }
  .card img {
    width: 40%;
  }
}
```
